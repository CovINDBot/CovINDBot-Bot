# CovINDBot-Bot

### Installation

```
git clone https://github.com/CovINDBot/CovINDBot-Bot.git
cd CovINDBot-Bot/
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
```

Create secret.ini file from example.secret.ini

To run the bot

```
python3 main.py
```
